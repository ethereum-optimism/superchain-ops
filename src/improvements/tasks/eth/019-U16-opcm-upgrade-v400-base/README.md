## Simulation

When simulating, ensure the logs say `Using script <your_path_to_superchain_ops>/superchain-ops/src/improvements/template/OPCMUpgradeV400.sol`.
Navigate to the correct task directory then run the simulate command.
```
cd src/improvements/tasks/eth/020-U16-opcm-upgrade-v400-base

just simulate-stack eth 019-U16-opcm-upgrade-v400-base base-nested base-council
```
